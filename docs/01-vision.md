# Vision

Omekan ist eine Community-zentrierte Eventplattform für Deutschland.
Fokus liegt auf kulturellen Communities (z. B. türkisch, griechisch),
mit Events, Künstlern und Veranstaltern.

## Ziel
- Events sichtbar machen
- Künstler hervorheben
- Veranstaltern Reichweite geben

## Monetarisierung (später)
- Partner-Organizer erhalten Tokens
- Tokens erhöhen Event-Sichtbarkeit (Promotion)
- Affiliate-Links zu externen Event-/Ticketseiten

## Scope V1
- Keine Ticketverkäufe
- Keine Social Features außer Bewertungen
- Keine App
- Keine KI
